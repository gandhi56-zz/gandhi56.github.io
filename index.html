Projects

The above program counts the number of nonnegative integers which are at most the input integer satisfying n+x = n xor x. I found this quite fascinating! Following are brief descriptions about some projects that I have done in class or just for fun. Visit my github to learn more about them!

  1. unixFS is a UNIX-based file system, written in C++, which stores data files persistently on disk, represented by a file. Features include:
    • Disk mounting
    • File/directory creation and deletion
    • Read/write file
    • Update buffer
    • File resize
    • Disk defragmentation

  2. mapReduce library is a programming model and a distributed computing paradigm for large-scale data processing, written in C++. It allows for applications to run tasks in parallel, making them scalable and fault-tolerant.

    To use the mapreduce library, developers must implement two callback functions, namely map and reduce.

    • The map function takes in filename as input and generates intermediate key/value pairs from the data stored in that file.
    • The reduce function processes a subset of intermediate key/value pairs, summarizing all values associated with the same key.

  3. lianshell is a standalone procedurally designed UNIX-based shell program, written in C++, that supports the following features:
    • Support for built-in commands including:
      • changing working directory
      • modifying environment variables
      • killing all subprocesses
    • Runs external programs from any specified location in the file system
    • Sequential execution of multiple processes
    • Redirection of output from a process to a file
    • Run processes in the background
    • Implement piping of output from one program to another
    • Handles signal interrupts, including SIGINT and SIGTSTP.
  4. Q3T is an implementation of Quantum Tic-tac-toe. This game integrates the ideas of superposition and entanglement in the classical game of Tic-tac-toe, written in Python 3.
  5. dragon is a single-header library of game playing algorithms integrable with any game written in the expected format. Currently, the project includes Nex, Hex, Reversi and Tic-tac-toe as example programs. This project is written in C++.
  6. EulerTikz is a visualization tool for discrete graphs which implements spectral layout algorithms for graph drawing.
